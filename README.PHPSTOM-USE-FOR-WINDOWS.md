# phpstrom 使用说明

## 分割窗口
Window -> Editor Tabs -> Split Verticlally

## 切换皮肤

```
Ctrl + ~
```

## 光标不移动滚屏

```
Ctrl + Up/Down
```

## 光标滚屏到屏幕中间

```
Ctrl + M
```

## 折叠一个嵌套

```
Ctrl + .
```

## 跳到嵌套的`{`/`}`

```
Ctrl + [ / Ctrl + ]
```

## 打开显示空格
settings -> 搜索`show whitespace`

## 快速左右滚屏
shift + 鼠标滚轮

## 返回上/下一次修改

```
Ctrl + Shift + Alt + Up/Down
```
